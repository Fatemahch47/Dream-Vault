# 🌙 Dream Vault

A Python-based Dream Management System that helps users store, organize, and track their dreams and life goals. This project was developed to practice Object-Oriented Programming (OOP), File Handling, and other core Python concepts.

---

## 📌 Project Description

Dream Vault allows users to save their dreams with details such as title, category, target year, motivational quote, notes, and progress. It also provides features to update progress, view favourite dreams, generate motivation reports, and save records to a file.

---

## ✨ Features

- ➕ Add New Dream
- 🔍 Search Dream
- 📋 Show All Dreams
- 📈 Update Dream Progress
- ❌ Delete Dream
- ❤️ Show Favourite Dreams
- 💬 Motivation Report
- 💾 Save Dreams to Text File
- 📖 Load Saved Dreams
- 🌟 Quote of the Day
- 📊 Total Dreams Counter
